## Tech Stack

**Server:** Node, Hono, JavaScript, TypeScript


## API Reference

#### Get validate 
takes the api key and return true or false if validated

```http
  GET /validate
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
